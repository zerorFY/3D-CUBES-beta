# 3D Block Builder

一个基于 Three.js 的 3D 方块搭建工具，支持桌面和移动设备。

## 在线体验

[点击这里体验](https://你的用户名.github.io/仓库名)

## 功能特性

- 🎨 **多色方块** - 4 种颜色可选
- 🧲 **智能吸附** - 26 方向顶点吸附算法
- 👆 **触屏支持** - 完美适配 iPad 和移动设备
- 🎮 **AutoCAD 风格控制** - 专业的 3D 导航体验
- 👁️ **透视模式** - 按住 T 键查看内部结构

## 控制说明

### 桌面端
- **左键点击** - 放置方块
- **Shift + 左键** - 删除方块
- **右键拖动** - 旋转视角
- **中键拖动** - 平移画面
- **滚轮** - 缩放
- **1-4 键** - 切换颜色
- **T 键（按住）** - 透视模式

### 移动端
- **单指点击** - 放置方块
- **双指捏合** - 缩放
- **双指拖动** - 平移画面

## 技术栈

- Three.js - 3D 渲染
- Vanilla JavaScript - 核心逻辑
- HTML5 + CSS3 - 界面

## 本地运行

直接打开 `index.html` 文件即可。
